# Atliq Sales Analytics Project

##  Overview

This project analyzes sales data for **Croma India (Atliq Hardware)** using SQL to generate insights on product performance and monthly sales trends.

---

## Objective

* Generate **monthly sales reports**
* Analyze **product-level performance**
* Enable further analysis in Excel/Sheets


## Dataset

* `dim_customer`
* `dim_product`
* `fact_sales_monthly`
* `fact_gross_price`


##  Tools Used

* SQL (MySQL)
* Excel 


##  Key Analysis

* Monthly sales trend
* Top-performing products
* Customer-level sales analysis

---

##  Key Insights

* Sales vary across months with peak periods
* Few products contribute major revenue
* Some products show low performance

---

##  Outcome

Built an end-to-end SQL project to transform raw data into meaningful business insights.
